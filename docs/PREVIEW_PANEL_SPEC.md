# Preview Panel 기능 명세서

> 최초 작성: 2026-03-29
> 원칙: 프리미어/캡컷/다빈치의 기본 기능을 100% 커버한 후, AI-Studio 고유 기능을 추가한다.
> 기존 사용자가 "이건 아마추어가 만든 거네"라고 느끼는 순간 다시 안 돌아온다.

## 1. 경쟁 제품 프리뷰 패널 기능 분석

### 1-1. Premiere Pro Program Monitor
- 재생 컨트롤: Play/Pause, JKL 셔틀, 프레임 단위 이동, 루프 재생
- 타임코드: HH:MM:SS:FF 표시, 직접 입력으로 이동
- In/Out 포인트: 설정, 이동, 범위 표시
- 재생 해상도: Full, 1/2, 1/4, 1/8, 1/16 드롭다운 (성능 조절)
- 줌/확대: Fit, 25%~400% 확대, 드래그로 패닝
- 안전 구역(Safe Margins): Action Safe (90%), Title Safe (80%), 4:3 Safe
- 가이드/그리드: 사용자 정의 가이드 라인, 스냅 기능
- 오버레이: 소스 타임코드, 마커 코멘트, 편집점 표시, 미디어 끝 표시
- 오버레이 프리셋: 저장/불러오기/내보내기
- Transform 핸들: 프리뷰에서 직접 이동/크기/회전 조작
- 스냅 정렬: 오브젝트 간 스냅, 중앙선 스냅, 가이드 스냅
- 프레임 내보내기: 현재 프레임을 스틸 이미지로 저장
- 프록시 토글: 원본/프록시 전환 버튼
- 비교 뷰: A/B 비교, 나란히 보기
- 전체 화면: 더블클릭 또는 단축키로 전체 화면 전환
- 외부 모니터 전송: Transmit to external monitor

### 1-2. DaVinci Resolve Viewer
- 안전 구역: Action Safe, Title Safe, Center 크로스헤어, Extents
- 사용자 정의 가이드: SNS 플랫폼별 가이드 (YouTube, Instagram, TikTok)
- 뷰어 오버레이: Transform, Crop, Dynamic Zoom, Composite, Retime
- 비디오 스코프: 히스토그램, 벡터스코프, 웨이브폼 (별도 패널 또는 오버레이)
- 줌: 마우스 휠 줌, Alt+스크롤 패닝
- 컬러 피커: 뷰어에서 직접 색상 추출
- 와이프(Wipe): 보정 전/후 비교
- LUT 프리뷰: 3D LUT 오버레이
- 마스크 오버레이: 파워 윈도우/매직 마스크 경계 표시

### 1-3. CapCut Desktop/Web
- 캔버스: 16:9, 9:16, 1:1, 4:5 등 비율 전환
- 룰러/가이드: 드래그로 가이드 생성, 스냅 정렬
- Transform 핸들: 8점 핸들 + 회전 핸들 + 중심점
- 스냅: 중앙선, 가장자리, 다른 오브젝트 간 스냅 (자석 표시)
- 텍스트 직접 편집: 프리뷰에서 더블클릭으로 텍스트 인라인 편집
- 배경 설정: 블러 배경, 단색 배경, 이미지 배경
- 재생 컨트롤: 간결한 Play/Pause, 앞/뒤 이동
- 해상도 전환: 원본 해상도 / 성능 모드

### 1-4. MasterSelects (WebGPU 브라우저 기반)
- WebGPU 캔버스: GPU 렌더링, 60fps
- Transform 핸들: 이동/크기/회전 직접 조작
- RAM Preview: 캐시된 프레임 즉시 재생
- 멀티 아웃풋: 여러 윈도우로 출력
- 슬라이스 마스크: 출력 영역 분할
- 코너 핀 워핑: 프로젝션 매핑용

## 2. AI-Studio 프리뷰 패널 전체 기능 명세

### 카테고리 A: 기본 기능 (경쟁 제품 100% 커버 — 이것 없으면 아마추어)

#### A-1. 재생 컨트롤
- Play/Pause (Space)
- 프레임 단위 이동 (←/→)
- JKL 셔틀 (역재생/정지/정방향, 연타 시 2x/4x/8x)
- 처음으로 (Home), 끝으로 (End)
- In 포인트 이동 (Shift+I), Out 포인트 이동 (Shift+O)
- 루프 재생 (In/Out 범위 또는 전체)
- 재생 속도 표시 (1x, 2x, 0.5x 등)

#### A-2. 타임코드
- HH:MM:SS:FF 형식 표시 (프로젝트 fps 기반)
- 클릭하여 직접 타임코드 입력 → 해당 시점으로 이동
- 현재 프레임 번호 표시 (토글)
- 총 길이 / 남은 시간 표시

#### A-3. In/Out 범위
- I 키로 In 포인트 설정
- O 키로 Out 포인트 설정
- In/Out 구간 시각적 표시 (프리뷰 타임라인 바에 하이라이트)
- 구간 길이 표시
- Alt+X로 In/Out 초기화

#### A-4. 재생 해상도 (성능 조절)
- 드롭다운: Full, 1/2, 1/4, 1/8
- 편집 중에는 1/4, 정지 시 Full 자동 전환 옵션
- GPU Tier에 따라 기본값 자동 설정 (Tier 1: Full, Tier 2: 1/2, Tier 3-4: 1/4)

#### A-5. 줌/확대
- Fit (프리뷰 영역에 맞춤) — 기본값
- 25%, 50%, 75%, 100%, 150%, 200%, 400%
- 마우스 휠로 줌 인/아웃
- 줌 시 드래그로 패닝
- 더블클릭으로 Fit 복귀

#### A-6. 안전 구역 (Safe Margins)
- Action Safe (90%) — 빨간색 점선
- Title Safe (80%) — 노란색 점선
- Center 크로스헤어 — 십자선
- 4:3 Safe (레터박스 내) — 파란색 점선
- SNS 안전 구역 (YouTube 자막 영역, TikTok 버튼 영역 등) — AI-Studio 고유
- 토글: 개별 on/off + 전체 on/off
- 색상/투명도 커스터마이즈

#### A-7. 가이드/그리드
- 그리드: 3x3 (삼등분법), 4x4, 사용자 정의
- 가이드라인: 수직/수평 드래그 생성, 위치 숫자 입력
- 가이드 잠금/삭제/전체 삭제
- 가이드에 스냅: 오브젝트 이동 시 자석처럼 달라붙음

#### A-8. Transform 핸들 (프리뷰 직접 조작)
- 8점 크기 조절 핸들 (모서리 + 변 중앙)
- 회전 핸들 (모서리 밖)
- 중심점 이동 (Alt+드래그)
- Shift 드래그: 비율 유지 크기 조절
- 수치 입력 패널: X, Y, W, H, Rotation, Scale 직접 입력
- 복수 레이어 동시 선택/이동

#### A-9. 스냅 정렬
- 중앙선 스냅 (수직/수평)
- 가장자리 스냅 (상/하/좌/우)
- 다른 오브젝트 가장자리 스냅
- 등간격 분배 가이드 (3개 이상 오브젝트 선택 시)
- 스냅 라인 시각적 표시 (마젠타색 가이드 라인)
- 스냅 on/off 토글 (자석 아이콘)

#### A-10. 프레임 내보내기
- 현재 프레임을 PNG/JPG로 저장
- 클립보드에 복사
- 해상도: 원본 / 프리뷰 크기 선택

#### A-11. 전체 화면
- 프리뷰 더블클릭 또는 ~ 키로 전체 화면
- ESC로 복귀
- 전체 화면에서도 재생 컨트롤 사용 가능 (마우스 오버 시 표시)

#### A-12. 캔버스 비율 전환
- 16:9 (가로 — YouTube)
- 9:16 (세로 — Shorts/Reels/TikTok)
- 1:1 (Instagram 피드)
- 4:5 (Instagram 세로)
- 4:3 (레거시)
- 사용자 정의 비율 입력
- 전환 시 기존 레이아웃 자동 조정 옵션

#### A-13. 배경 설정
- 검정 (기본)
- 체커보드 (투명 표시)
- 사용자 정의 색상
- 소스 블러 배경 (세로 영상의 가로 배경)

#### A-14. 텍스트 인라인 편집
- 프리뷰에서 텍스트 클립 더블클릭 → 직접 텍스트 수정
- 드래그로 위치 변경
- Properties 패널과 실시간 동기화

### 카테고리 B: 프로 기능 (중급 이상 사용자 — 경쟁 우위)

#### B-1. 오버레이 시스템
- 소스 타임코드 오버레이
- 클립 이름 오버레이
- 마커 코멘트 오버레이
- 편집점 표시 (클립 시작/끝 프레임에 꺾쇠 표시)
- 미디어 끝 표시 (보라색 줄)
- 오버레이 위치: 4개 코너 + 상/하 중앙
- 오버레이 프리셋 저장/불러오기
- 오버레이 투명도/텍스트 크기 조절
- 재생 중 오버레이 표시 on/off

#### B-2. 비교 뷰
- A/B 비교: 이펙트 적용 전/후 나란히 보기
- 와이프(Wipe): 슬라이더로 좌/우 분할 비교
- 블렌드: 50% 투명도로 겹쳐 보기
- 비교 대상 선택: 이전 프레임 / 이펙트 없는 원본 / 다른 클립

#### B-3. 컬러 피커
- 프리뷰에서 클릭하여 색상값 추출 (RGB/HEX/HSL)
- 추출한 색상을 클립보드에 복사
- 추출한 색상을 텍스트/배경에 바로 적용

#### B-4. 비디오 스코프 (GPU 가속)
- 히스토그램 (RGB + Luma)
- 웨이브폼 모니터
- 벡터스코프
- 프리뷰 하단 또는 별도 패널에 표시
- 실시간 업데이트 (재생 중에도)

#### B-5. 프록시 토글
- 원본/프록시 전환 버튼
- 프록시 미생성 시 자동 생성 제안
- 현재 모드 표시 (프록시/원본)

### 카테고리 C: AI-Studio 고유 기능 (차별화 — 기존 툴에 없는 것)

#### C-1. AI 구도 추천
- 현재 프레임 분석 → 삼등분법, 황금비, 대칭 등 구도 제안
- "이 피사체를 여기로 이동하면 더 나은 구도" 가이드 오버레이
- 한 클릭으로 AI 추천 적용

#### C-2. SNS 플랫폼별 안전 구역 프리셋
- YouTube: 자막 영역, 종료 화면 영역, 워터마크 영역
- TikTok: 하단 설명, 우측 버튼, 프로필 아이콘 영역
- Instagram Reels: 하단 캡션, 좌측 프로필
- Shorts: YouTube Shorts 고유 UI 영역
- 커스텀 플랫폼 추가 가능
- 클릭 한 번으로 전환 → 해당 플랫폼 안전 구역 + 비율 동시 적용

#### C-3. 스크립트 연동 프리뷰
- AI 스크립트(대본)와 타임라인 동기화
- 프리뷰에서 현재 대본 문장 하이라이트 오버레이
- 대본 클릭 → 해당 시점으로 이동
- 워드 타이밍과 동기화된 자막 실시간 프리뷰

#### C-4. 프리셋 추출/저장/인라인 미리보기
- 현재 클립의 Transform + Effect + Text 설정을 "프리셋"으로 저장
- 프리셋 라이브러리: 썸네일 미리보기 + 카테고리 분류
- 드래그 앤 드롭으로 다른 클립에 적용
- "이전 성공 영상" 프리셋 자동 추출: 과거 내보내기 완료 프로젝트의 설정을 자동 수집
- 인라인 미리보기: 프리셋 위에 마우스 올리면 현재 클립에 적용된 결과를 프리뷰에 실시간 표시

#### C-5. AI 이펙트 미리보기
- 이펙트 패널에서 이펙트 위에 마우스 올리면 → 프리뷰에 해당 이펙트 적용 결과 실시간 표시
- 적용하지 않아도 미리 볼 수 있음 (hover preview)
- "추천 이펙트" AI 제안: 현재 클립 분석 → 어울리는 이펙트 3~5개 추천

#### C-6. 멀티 비율 동시 프리뷰
- 16:9 + 9:16 + 1:1을 동시에 작은 창으로 표시
- "이 편집이 세로에서 어떻게 보이는지" 실시간 확인
- Auto-Reframe AI: 가로 영상에서 피사체 추적하여 세로 크롭 자동 제안

#### C-7. 터치/펜 지원
- 펜 태블릿 압력 감지: 마스크 그리기 시 페더링 연동
- 멀티 터치: 핀치 줌, 투핑거 회전
- 제스처: 세 손가락 스와이프 = 타임라인 스크러빙

## 4. 기술 구현 매핑

| 기능 | 렌더링 엔진 | GPU 활용 |
|------|-------------|----------|
| 비디오 프레임 표시 | WebGPU importExternalTexture | zero-copy GPU |
| Transform 핸들 오버레이 | WebGPU render pass (별도 레이어) | GPU |
| 안전 구역/가이드/그리드 | WebGPU render pass (라인 렌더링) | GPU |
| 텍스트 오버레이 | WebGPU + SDF 텍스트 렌더링 | GPU |
| 비디오 스코프 | WebGPU compute shader | GPU |
| 이펙트 적용 | WebGPU compute shader | GPU |
| 스냅 라인 표시 | Canvas 2D overlay (DOM 위) | CPU (경량) |
| 타임코드/오버레이 텍스트 | DOM (CSS) overlay | CPU (경량) |
| AI 구도 분석 | ONNX Runtime WebGPU | GPU |
| 폴백 (Tier 3-4) | Canvas 2D / WebGL2 | CPU / 경량 GPU |
